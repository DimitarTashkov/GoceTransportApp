## 2. Описание на Проекта (About the Project)
* Напиши силно "Elevator pitch" представяне. Какво е това? Това е мултитенант (Multi-tenant) SaaS платформа за автобусни превозвачи и пътници.
* Обясни накратко как решава два проблема: дава на пътниците лесен начин да търсят и купуват билети, а на превозвачите – пълна система за управление на автопарк, персонал и разписания.

## 3. Основни Функционалности (Key Features)
Раздели ги на логически групи (използвай емоджита за прегледност):
* **👤 За Пътници:** Търсене на маршрути, купуване на билети (QR/PDF), локализация (БГ/EN флагове), табло за управление (User Dashboard), следване на любими превозвачи.
* **🏢 За Превозвачи (Организации):** Управление на автопарк (Vehicles), шофьори (Drivers) и маршрути (Routes). Генериране на разписания. Organization Dashboard с метрики.
* **⚙️ Системни/Админ:** Абонаментни планове (Membership Tiers), които лимитират ресурсите, Identity система за роли, автоматично сийдване на базата (Master Seeder).

## 4. Технологичен Стек (Tech Stack)
* Изброй технологиите: C# / ASP.NET Core MVC, Entity Framework Core, SQL Server.
* Фронтенд: Bootstrap, HTML5/CSS3, JavaScript.
* Инструменти: Playwright (за E2E автоматизирано тестване).

Направи го стъпка по стъпка (Step-by-step), все едно обясняваш на нов програмист в екипа ти:
